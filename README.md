# Netflix Data Analysis 📊

## 📌 Project Overview
This project analyzes Netflix dataset using Python.

## 🛠️ Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## 🔍 Key Features
- Data filtering (Movies vs TV Shows)
- GroupBy analysis (Country, Year)
- Data visualization (Bar, Pie, Line charts)

## 📈 Insights
- Compared Movies vs TV Shows
- Identified top countries producing content
- Analyzed release trends over years

## 🚀 How to Run
1. Install dependencies:
   pip install pandas matplotlib seaborn
2. Run:
   python netflix_analysis.py
